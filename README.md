# 🧾 Sistema de Vendas Simples (Vanilla JS)

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema de vendas simples (estilo mercadinho), construído inicialmente utilizando apenas:

* HTML
* CSS
* JavaScript puro (Vanilla JS)

O objetivo não é apenas criar uma aplicação funcional, mas sim **entender profundamente os fundamentos do desenvolvimento web moderno**, antes de utilizar frameworks como React ou Node.js.

---

## 🎯 Objetivo Principal

Construir uma base sólida em JavaScript, compreendendo na prática:

* Manipulação do DOM
* Controle de estado manual
* Organização de código
* Estruturação de dados
* Persistência no navegador

Após essa etapa, o projeto será **refatorado utilizando tecnologias modernas**, permitindo comparar:

> “como era feito manualmente” vs “como frameworks resolvem isso”

---

## 🧠 Filosofia do Projeto

Este projeto segue uma abordagem intencional:

> **aprender primeiro os fundamentos, depois abstrações**

Em vez de começar com frameworks, a ideia é:

1. Resolver problemas reais com JavaScript puro
2. Entender as dificuldades desse processo
3. Migrar para ferramentas modernas com consciência

---

## 🧩 Funcionalidades (Planejadas)

### 🛒 Produtos

* Cadastro de produtos
* Listagem
* Edição
* Remoção

### 🧺 Carrinho

* Adição de produtos
* Remoção de itens
* Cálculo de total

### 💳 Vendas

* Finalização de venda
* Formas de pagamento:

  * Dinheiro
  * Pix
  * Cartão
  * Fiado

### 💾 Persistência

* Armazenamento com `localStorage`

---

## 🧱 Etapas de Desenvolvimento

O projeto será desenvolvido em fases:

### 1. Estrutura inicial

Organização do projeto e criação do layout base

### 2. CRUD de produtos

Manipulação completa de dados

### 3. Carrinho e estado

Controle manual de estado da aplicação

### 4. Persistência

Uso de `localStorage`

### 5. Refatoração

Melhoria da organização e estrutura do código

### 6. Migração para React

Reescrita do projeto com foco em abstração e produtividade
